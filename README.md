# Khaled Eissa Portfolio

Personal portfolio website with:
- Light colorful UI
- Featured case studies from selected GitHub projects
- Live GitHub repositories loaded from API
- Live profile stats (repos, stars, followers)

## Featured repositories
- Sign-Language-ML
- QR-Cart-Shopping-System
- Health-Center-DB
- HMK-Charity

## Files
- `index.html` - content and sections
- `styles.css` - theme and responsive layout
- `script.js` - GitHub API loading and dynamic repo cards

## Run locally
Open `index.html` in a browser.

## Quick edits
- Profile image path is currently: `peronal profile picture.jpeg`
- Resume line in contact section is set to "Available on request"
- GitHub username default is set in `script.js` as `DEFAULT_USERNAME`
